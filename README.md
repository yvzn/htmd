## How ?

This is a static HTML website, packaged with [ViteJS](https://vitejs.dev/)

## Run locally

Clone the repository, then:

```bash
npm install
npm run dev
```

And open http://localhost:5173/ in browser of choice.

<kbd>Ctrl + C</kbd> to stop.
